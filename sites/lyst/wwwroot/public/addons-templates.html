How to Use Our Templates

Using the grid you can build tons of layouts. Without using a single line of CSS outside of what comes with Foundation, we've created thirteen layouts for you to start from and build on. It's dead simple to use these simple HTML templates, just grab the code and drop it between the <body> tags of a Foundation page.
